Gen 6 pokemon in 3d! (without the glasses)

This was the generation that i was looking forward to because I was interested in how they would use the 3d effect in the end i was happy with them only using it in battle because it would have been too much if it was also on everything else I feel like this gen was fun with the 3d effect and a return to from with game freak.

Pokemon X and Y

Pokemon x and y where the first games which 3d effects from the 3ds which were quite fun at the time they introduced mega evolutions which were a very fun concept but before they were released i was worried about the mega evolutions but luckily with only 1 per battle they were well balanced my only complaint was the gap between the first and second gyms (it felt like it was taking forever).

Pokken tournament

Pokken tournament was an arcade pokemon game (in japan only) in the rest of the world it was released on the wii U.It is a fighting game made by namco bandai in a similar mold to tekken where the pokemon move around the arena and attack.I have only played the game once it was a fun game but it felt more like tekken and less like pokemon but I do think with minor tweaks it could be a great series.
